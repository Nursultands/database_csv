# database_csv
easy way to create database of csv
Hi everyone! I am a beginner proggramer! Here I created a database of csv with help Object-oriented programming. I used Inheritance and Association and etc. 
Here I used class School as container in which I collected object of class Human, Student and Teacher.
Task of function get_report() it write information about school, teachers and pupils in csv file.
Task of function load_csv() load dates from already existing csv file to attributes of class Student and Teacher,in order in the future while adding a new data about
Students and Teacher without losing an old data from csv.
